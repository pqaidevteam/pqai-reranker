# PQAI Reranker

Service for ranking documents.

## Routes

| Method | Endpoint  | Comments                                                           |
| ------ | --------- | ------------------------------------------------------------------ |
| `GET`  | `/rerank` | Compares a set of documents with a query and sort them accordingly to similarity with the query |
